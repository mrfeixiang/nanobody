05 IgBLAST 사용법 & BCR→SHM 파이프라인

IgBLAST 실행 방법

# 1. Germline DB 준비
makeblastdb -parse_seqids -dbtype nucl \
  -in imgt_camelid_V.fasta -out V
makeblastdb -parse_seqids -dbtype nucl \
  -in imgt_camelid_D.fasta -out D
makeblastdb -parse_seqids -dbtype nucl \
  -in imgt_camelid_J.fasta -out J

# 2. IgBLAST 실행
igblastn -germline_db_V V \
  -germline_db_D D -germline_db_J J \
  -organism camel -domain_system imgt \
  -query input.fasta \
  -outfmt 19 -out results.tsv

# 3. SHM 계산
# v_identity 필드에서:
# SHM rate = 1 - v_identity

VHH에 대한 시사점
VHH는 ~360 nt로 비교적 짧지만, SHM 검출을 위해 단일 염기 수준의 정확도가 필수. PacBio HiFi가 최적이나, Illumina 2x250bp도 VHH 전장 커버 가능. Nanopore는 호모폴리머 오류로 CDR 호출 정확도 주의 필요.

09 PacBio Alignment & FLAIRR-seq

PacBio CCS/HiFi 원리

1. 헤어핀 어댑터로 원형 템플릿 생성
2. 폴리머라제가 여러 번 원형 통과
3. 각 패스는 독립적 오류 → 합의 시퀀스
4. 결과: Q30+ (>99.9%) 정확도, 최대 25 kb

관련 도구

lima바코드 분리
pbmm2HiFi 리드 정렬 (minimap2 래퍼)
minimap2장편 리드 범용 정렬
IgBLASTVDJ 주석

FLAIRR-seq 워크플로우

FLAIRR-seq (Full-Length AIRR Sequencing)
5' RACE 표적 증폭 + PacBio HiFi로 ~2 kb 면역글로불린 중쇄 전장 시퀀싱. IGHV + IGHD + IGHJ + IGHC 영역 동시 분석. 단편 리드 AIRR-seq (≤600 nt)의 한계 극복.
VHH에서 PacBio의 이점
VHH는 ~400 bp로 비교적 짧아 PacBio의 장점(장편 리드)이 제한적. 그러나:
  • VHH + 상수영역 전장 동시 분석 가능
  • IgG1 vs IgG2/3 아이소타입 구분 단일 리드
  • 99.9%+ 정확도로 SHM 검출 최적
