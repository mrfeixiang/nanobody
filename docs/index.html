나노바디란?
나노바디(VHH)는 낙타과 동물(낙타, 라마, 알파카)에서 발견되는 중쇄 항체(HCAb)의 가변 도메인입니다. ~15 kDa로, 자연에서 발견되는 가장 작은 항원 결합 단편이며, 기존 항체의 약 1/10 크기입니다.

08트러블슈팅 기록

분석 과정에서 실제로 발생한 문제와 해결 방법.

1. NCBI SSL 연결 오류
curl: (56) OpenSSL SSL_read: error:0A000126

원인: OpenSSL 3.6.2(conda)와 NCBI TLS 설정 비호환.
해결: ENA API 대체 사용 또는 curl -k로 SSL 검증 우회.
2. VHH 추출 실패 — 0개 서열 발견
ORF 탐색이 120 aa VHH 대신 6-109 aa 단편만 반환.

원인: FR2 영역의 amber 종결 코돈(TAG → *)이 ORF를 절단.
해결: ORF 탐색 대신 보존 모티프 검색(QVQL...WGQG) 사용. 내부 *를 X로 치환.
3. IgBLAST D/J 데이터베이스 누락
BLAST Database error: No alias or index file found for [human_gl_D]

원인: conda IgBLAST에 V 유전자만 포함, D/J BLAST 데이터베이스 없음.
해결: D/J germline FASTA를 수동 생성하고 makeblastdb로 BLAST DB 구축.
4. IMGT 데이터베이스 다운로드 실패
curl이 IMGT에서 FASTA 대신 HTML을 반환.

원인: IMGT 웹사이트가 브라우저 기반 접근을 요구.
해결: IgBLAST 내장 internal_data의 V 유전자 사용, D/J 서열 수동 준비.
